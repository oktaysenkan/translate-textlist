## Usage 

Create `list.txt` same folder with Translate.exe and run.

**list.txt**
```
computer
red
book
software
```
**word.json**
```json
[{"English":"computer","Turkish":"bilgisayar"},{"English":"red","Turkish":"kırmızı"},{"English":"book","Turkish":"kitap"},{"English":"software","Turkish":"yazılım"}]
```
